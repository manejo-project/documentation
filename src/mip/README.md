# Pragas, doenças e inimigos naturais

<p align="justify">
Para adicionar uma ocorrência múltipla basta selecionar uma unidade de referência no menu lateral esquerdo, depois clicar em "Nova ocorrência de pragas/doenças/inimigos naturais", após alguns dados serão necessários sendo eles a data da amostragem, o estádio da cultura e a desfolha. 
</p>
<p align="justify">
Caso haja pragas no plantio encontrar a nome correspondente deste na lista e inserir a média encontrada, o mesmo caso haja doenças e inimigos naturais. Lembrando que para adicionar uma nova ocorrência é necessário ter criado uma safra antes. Abaixo um tutorial em vídeo de como adicionar uma ocorrência múltipla.
</p>

## Nova ocorrência de Pragas

<p align="justify">
Para adicionar apenas a ocorrência de pragas basta seguir os passos do primeiro item, será necessário configurar o item "Flutuação das Pragas", neste passo uma lista será disponibilizada e será preciso encontrar a praga que deseja relatar e a média encontrada no plantio. ## Nova ocorrência de Doenças

<p align="justify">
Para adicionar apenas a ocorrência de doenças basta seguir os passos do primeiro item, será necessário configurar o item "Doenças", neste passo uma lista será disponibilizado duas opções Largarta com Nomuraea (doença branca) e Lagarta com Boculovirus (doença preta), após insira a quantidade a média encontrada.
