    # Pragas, Doenças e Inimigos Naturais

    Nesta seção iremos aprender como adicionar uma ocorrência de pragas, doenças e inimigos naturais que podem ocorrer em uma safra.

    Tópicos a serem configurados:

    • Dados da Amostragem
    • Flutuação de Pragas
    • Doenças
    • Inimigos Naturais
    • Revisão
    • Vídeos Extras

    Abaixo um tutorial em vídeo de como configurar o Gerenciamento da Safra.

    Alerta

    Uma ocorrência só pode ser adicionada se o Produtor, Gerenciamento das URs e o Gerenciamento das Safras já estiverem cadastrados.

    # Dados da Amostragem

    • Passo 1. No canto esquerdo da página há um menu lateral, há uma lista em "Minhas Unidades de Referência", clique na UR que deseja adicionar a ocorrência.
    • Passo 2. Selecione a data da coleta dessa anotação de campo no calendário, para trocar de mês basta clicar nas setas que se encontram ao lado direito e esquerdo de mês e ano.
    • Passo 3. Selecione o Estádio da Cultura na lista ou pesquise ca caixa de texto, clique no estádio desejado, quando este estiver selecionado a cor de fundo do estádio ficará azul.
    • Passo 4. Digite o valor da desfolha em porcentagem na caixa de texto branca associada a "% de Desfolha (em números inteiros)".
    • Passo 5. Clique no botão azul "Próximo", localizado no canto inferior direito.

    # Flutuação de Pragas

    Está seção só deve ser preenchida caso haja pragas encontradas no plantio. Caso contrário pule para o passo 2.

    • Passo 1. Nesta página há uma lista de pragas, encontre a praga da qual deseja adicionar nesta ocorrência. Após encontrar digite a média encontrada na terceira coluna da lista porém na linha que corresponde a praga que deseja adicionar. Faça isso para todas as pragas que houver encontrado no plantio.
    • Passo 2. Clique no botão azul "Próximo", localizado no canto inferior direito.

    # Doenças

    Está seção só deve ser preenchida caso haja doenças encontradas no plantio. Caso contrário pule para o passo 2.

    • Passo 1. Nesta página há duas doenças (Lagarta com Nomuraea e Lagarta com Baculovirus), encontre a doença da qual deseja adicionar nesta ocorrência. Se há alguma dessas duas doenças no plantio digite a média encontrada na terceira coluna da lista porém na linha que corresponde a doença que deseja adicionar.
    • Passo 2. Clique no botão azul "Próximo", localizado no canto inferior direito.

    # Inimigos Naturais

    Está seção só deve ser preenchida caso haja Inimigos Naturais encontradas no plantio. Caso contrário pule para o passo 2.

    • Passo 1. Nesta página há uma lista de inimigos naturais, encontre o inimigo da qual deseja adicionar nesta ocorrência. Após encontrar digite a média encontrada na terceira coluna da lista porém na linha que corresponde ao inimigo que deseja adicionar. Faça isso para todos os inimigos que houver encontrado no plantio.
    • Passo 2. Clique no botão azul "Próximo", localizado no canto inferior direito.

    # Revisão

    Este passo é muito importante, revise todos os dados e somente após isso clique em "Salvar" que se encontra no canto superior esquerdo, se precisar alterar alguma informação clique em "Anterior" no canto inferior esquerdo e volte até a página que deseja fazer uma alteração. Caso deseje cancelar está anotação de campo é só clicar em "Cancelar" este se encontra ao lado de "Salvar".
