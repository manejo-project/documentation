    # Envio de Fotos

    O envio de fotos permite o envio de fotos de pragas, doenças e inimigos naturais.

    Alerta

    Só é possível enviar no máximo 10 fotos por envio. Cada foto deve ter no máximo 5MB. São aceitas apenas fotos GIF, PNG ou JPG.

    Abaixo um tutorial em vídeo de como enviar fotos.

    # Passos para Enviar Fotos

    Alerta

    Você pode fazer quantos envios quiser. Basta repetir os procedimentos 1 a 4 quantas vezes for necessário.

    • Passo 1. No canto esquerdo da página há um menu lateral. Selecione a opção "Envio de Fotos".
    • Passo 2. Clique no botão azul "+ Selecionar Fotos" este se encontra no canto superior esquerdo.
    • Passo 3. Note que uma aba será aberta e nela encontre as fotos que estão em seu computador. Selecione de uma a 10 fotos e depois clique em abrir.
    • Passo 4. Espere as fotos carregarem e quando finalizar uma mensagem será exibida "Sucesso Fotos Enviadas" localizado no topo da página.
